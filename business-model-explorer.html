üîó Understanding Network Effects

Metcalfe's Law: The value of a network is proportional to the square of the number of users.

Examples:

  • Social Networks: Facebook, LinkedIn - more friends = more value
  • Communication: WhatsApp, Zoom - value increases with more contacts
  • Marketplaces: eBay, Uber - more buyers attract more sellers and vice versa
